# Plume

一个轻量、快速、优雅的静态博客生成器。

[English](README.md) | 中文

- 🚀 **极速加载** - 无需加载多余的 JavaScript 框架

- 🎨 **简洁设计** - 简单易读

- 📦 **归档功能** - 隐藏旧文章但保持可访问
- �📱 **响应式设计** - 在所有设备上都有良好表现
- ♿ **无障碍访问** - 遵循 Web 无障碍指南

## 使用方法

1. 在 `posts/` 目录下创建 Markdown 文件
2. 运行构建命令生成 HTML

```bash
npm run build
```

生成的 HTML 文件将输出到 `public/` 目录。

## 文章格式

每篇文章需要在顶部包含 YAML front matter：

```markdown
---
title: 文章标题
date: 2025-10-27
modified: 2025-10-27  # 可选：最后修改时间
tags: [标签1, 标签2]  # 可选：文章标签
math: true            # 可选：启用数学公式
archived: true        # 可选：归档文章（不在首页显示）
---

## 这是标题

文章内容从这里开始...
```

### Front Matter 字段说明

- `title` (必需): 文章标题
- `date` (必需): 发布日期 (YYYY-MM-DD 格式)
- `modified` (可选): 最后修改日期
- `tags` (可选): 文章标签数组
- `math` (可选): 设为 `true` 启用 MathJax 数学公式渲染
- `archived` (可选): 设为 `true` 将文章归档（不在首页显示，但仍可访问）

## 功能特性

### 📐 数学公式

在 front matter 中设置 `math: true` 启用数学公式：

行内公式：`$E = mc^2$`

块级公式：
```
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### 🏷️ 标签系统

在 front matter 中添加标签：

```yaml
tags: [JavaScript, Tutorial, Web Development]
```

系统会自动生成：
- 每个标签的单独页面
- 所有标签的索引页面
- 文章中的标签链接

### 📦 归档功能

要归档某篇文章（不在首页显示但保持可访问）：

```yaml
archived: true
```

归档的文章会：
- 从首页列表中移除
- 在归档页面显示
- 仍然可以通过直接链接访问
- 仍然出现在标签页面中

### 📑 自动目录

系统自动为每篇文章生成目录，包含：
- 所有标题（h1-h6）
- 可点击的锚点链接
- 嵌套结构
- 当前章节高亮（滚动时）

### 🗑️ 自动清理

构建时会自动删除：
- 已删除文章对应的 HTML 文件
- 不再使用的标签页面
- 其他过时的文件
